# TransAI v2.5.5

## Анализ проектов цифровой трансформации Московской области

Дашборд для управления трансформационными проектами: загрузка данных из XLSX, валидация, AI-анализ через DeepSeek, хранение в MySQL.

## Стек

- **Frontend:** React 19 + TypeScript + Vite 7 + Tailwind CSS + Framer Motion + Recharts
- **Backend:** PHP 8.2 (REST API)
- **Database:** MySQL 8.0 (`trans_ai_bd`)
- **AI:** DeepSeek API

## Возможности

- Загрузка 3 XLSX файлов (РМ, ДБ, Аналитика ИИ)
- Автовалидация структуры каждого файла
- Автосохранение в MySQL с soft-delete
- AI-анализ обоснованности проектов через DeepSeek
- Блок "Рекомендации ИИ по внедрению" с фильтрацией
- 7 категорий вердиктов: рекомендован, однозначно, социальная направленность, с изменениями, не рекомендован, не рекомендуется, нет данных
- Динамическая ширина блоков (flex), кликабельная фильтрация проектов
- Экспорт enriched RM и полного отчета в XLSX

## Структура БД

- `projects` — проекты с `ai_verdict` и `ai_reasoning`
- `upload_batches` — версии загрузок
- `ai_analysis` — данные ИИ-анализа (key-value)
- `ai_reasoning` — результаты DeepSeek анализа

## Установка

1. Распакуйте `TransAI-source.zip` в корень сайта
2. Создайте БД `trans_ai_bd` и выполните `api/schema.sql`
3. Настройте `.env` (DB_* и DEEPSEEK_API_KEY)
4. Убедитесь что PHP выполняется в папке `api/`

## Промпт DeepSeek (v2.5.5)

Роль: руководитель цифровой трансформации в регионе. Формат ответа: "Оценка проекта" + "Краткое обоснование оценки" с конкретными цифрами.
